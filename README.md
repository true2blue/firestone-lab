# **Firestone-Lab**

A tool for test

---
## **Prerequisites**
- Python 3.12.4 installed.

## **Setup Instructions**
### install virtualenv
1.  pip install virtualenv
### create new env
2.  virtualenv venv
### activate the env
3.  cd venv/Scripts
    source ./activate
### install dependencies
4.  pip install -r requirements.txt

## **Libraries**

1. AKshare + AKtools (data)
2. ta-lib (indicator)
3. Backtesting.py (backtesting)
