以下是 **Backtesting.py** 回测结果中 `_trades` 表格列名的 **中文解释**，结合交易策略回测的上下文：

| 列名 (Column Name)       | 中文解释                                                                 |
|--------------------------|------------------------------------------------------------------------|
| **Size**                 | 交易头寸大小（例如：买入/卖出的股票数量、合约数量）。                          |
| **EntryBar**             | 入场时的 **K线柱序号**（例如：在历史数据的第100根K线入场）。                   |
| **ExitBar**              | 出场时的 **K线柱序号**（例如：在第120根K线平仓）。                            |
| **EntryPrice**           | 入场价格。                                                               |
| **ExitPrice**            | 出场价格。                                                               |
| **SL**                   | 止损价格（Stop Loss）。若未触发止损，可能显示为`NaN`。                       |
| **TP**                   | 止盈价格（Take Profit）。若未触发止盈，可能显示为`NaN`。                      |
| **PnL**                  | 该笔交易的 **盈亏金额**（Profit and Loss，正值表示盈利，负值表示亏损）。        |
| **ReturnPct**            | 该笔交易的 **收益率百分比**（基于头寸大小的百分比回报）。                       |
| **EntryTime**            | 入场时间戳（具体日期时间）。                                               |
| **ExitTime**             | 出场时间戳。                                                              |
| **Duration**             | 持仓持续时间（单位：K线柱数量 或 时间单位，取决于回测设置）。                   |
| **Tag**                  | 交易的 **标签/备注**（可选字段，用于标记策略名称、类型或其他自定义信息）。        |
| **Entry_SMA(C,10)**       | 入场时 **10周期收盘价的简单移动平均线（SMA）** 的值。                          |
| **Exit_SMA(C,10)**        | 出场时 **10周期收盘价的简单移动平均线（SMA）** 的值。                          |
| **Entry_SMA(C,20)**       | 入场时 **20周期收盘价的简单移动平均线（SMA）** 的值。                          |
| **Exit_SMA(C,20)**        | 出场时 **20周期收盘价的简单移动平均线（SMA）** 的值。                          |

---

### 关键说明：
1. **SMA(C, N)**：  
   - `C` 表示使用 **收盘价（Close Price）** 计算移动平均线。  
   - `N` 是移动平均的周期（例如：`SMA(C,10)` 表示10日收盘价均线）。

2. **用途**：  
   - 这些列（如`Entry_SMA(C,10)`）通常用于验证策略逻辑。例如，在均线交叉策略中，可以检查入场/出场时的均线值是否符合预期。

3. **示例场景**：  
   - 如果策略规则是“当价格上穿10日均线时买入”，则 `Entry_SMA(C,10)` 应接近入场时的10日均线值，且 `EntryPrice` 应高于该值。

4. **其他常见列**（若存在）：  
   - `TradeType`：交易类型（如`Long`/`Short`）。  
   - `MaxDrawdown`：该交易的最大回撤。
